이미지 영역지정 하이퍼링크 설정

방법

1. 이미지 URL을 입력 후 이미지 로드 버튼을 클릭 하세요.
2. Spot 생성 버튼을 클릭하여 새로운 Spot영역을 생성할 수 있습니다.
(여러개 생성시는 생성하고자 하는 갯수만큼 클릭)
3. 하이퍼 링크 주소와 대체 텍스트를 기재 하세요.
4. 이미지 위에 생성된 맵 영역을 드래그하여 원하는 위치로 드래그 하세요.
5. 맵영역 크기 설정은 박스 귀퉁이를 드래그 하여 설정할 수 있습니다.
6. 코드 생성 버튼을 클릭하면 맵 코드를 생성 할 수 있습니다.
7. 미리보기 버튼 클릭하면 링크된 내용의 미리보기 할 수 있습니다.
